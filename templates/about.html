  About the Project

FaceID Recognition System
A modern, web-based face recognition

Sistem pengenalan wajah real-time yang menggabungkan computer vision dan machine learning — dibangun dengan Mediapipe, face_recognition (128-D embeddings), dan SVM.

Project Overview

Proyek ini dirancang untuk menghadirkan identifikasi wajah secara real-time melalui antarmuka web yang sederhana. Menggunakan kombinasi Mediapipe untuk deteksi wajah, face_recognition untuk ekstraksi embedding 128 dimensi, dan Support Vector Machine (SVM) untuk klasifikasi yang akurat.

Seluruh sistem berjalan ringan di CPU, mudah dipasang, serta cocok untuk skenario demo, edukasi, maupun prototipe produk.

Core Technologies

Mediapipe

Deteksi wajah real-time dan landmark tracking langsung di browser.

face_recognition

Ekstraksi embedding wajah 128-D yang robust (berbasis dlib).

SVM Classifier

Klasifikasi identitas berbasis SVM yang cepat dan stabil.

Flask Backend

API ringan untuk pelatihan, identifikasi, dan manajemen dataset.

How It Works

  1. 1
    Face Detection

    Mediapipe menganalisis frame video dan mendeteksi wajah secara real-time.

  2. 2
    Feature Extraction

    Library face_recognition mengekstrak embedding 128-D dari wajah terdeteksi.

  3. 3
    SVM Classification

    Embedding dimasukkan ke model SVM yang telah dilatih untuk memprediksi identitas.

  4. 4
    Result Display

    Nama/label ditampilkan di overlay video. Jika jarak > threshold, ditandai sebagai Unknown.

Key Features

  • Real-time face detection (Mediapipe)
  • 128-D facial embeddings (face_recognition)
  • SVM with tunable unknown threshold
  • Works smoothly on CPU
  • Web-based interface & overlay
  • Simple registration → training workflow
  • Lightweight API (Flask, JSON)
  • Easy to deploy & extend

Technology Stack

Frontend

  • HTML, CSS
  • Vanilla JS (Mediapipe on client)

Backend

  • Flask (Python)
  • face_recognition
  • scikit-learn (SVM)
  • OpenCV
